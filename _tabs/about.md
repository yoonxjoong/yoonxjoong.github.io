안녕하세요, 백엔드 개발자 **yoonxjoong**입니다.

Spring / JPA를 중심으로 서버 개발을 공부하고 있고, 최근에는 MSA 환경에서 데이터 정합성을 지키는 방법(아웃박스 패턴 등)에 관심을 두고 프로젝트를 진행하고 있습니다. 그 외에도 자료구조와 알고리즘, 클린 코드, Kubernetes 같은 인프라까지 넓게 공부한 내용을 기록하고 있습니다.

## 이 블로그에서 다루는 내용

- **Backend**: Spring Boot, JPA, MSA, 아웃박스 패턴
- **Database**: 트랜잭션, 격리 수준, 지연/즉시 로딩 등 DB 이론
- **자료구조 & 알고리즘**: LinkedList, Stack/Queue, Tree, Trie, DFS/BFS, 코딩 테스트 풀이
- **Infra**: Kubernetes, AWS, Git 브랜치 전략
- **트러블슈팅**: 개발하며 겪은 문제와 해결 과정

정리하며 배우는 과정을 기록하는 공간이라, 틀린 내용이나 더 나은 방법이 있다면 언제든 알려주세요.
